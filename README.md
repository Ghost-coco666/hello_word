# Site Vitrine — hello_word

Ceci est un petit site vitrine de démonstration basé sur `page_principale.html`.

Prévisualisation locale

1. Ouvrez le dossier `hello_word` avec VS Code.
2. Installez l'extension Live Server (ritwickdey.LiveServer).
3. Faites un clic droit sur `page_principale.html` et choisissez "Open with Live Server".

Fichiers principaux

- page_principale.html — page d'accueil complète
- page_deco.css — styles (responsive)
- page_main.js — script léger pour menu mobile et formulaire

Personnalisation

- Remplacez les textes, ajoutez images dans le dossier et ajustez les couleurs dans `page_deco.css`.
